So funktioniert Deine Entwicklnugsumgebung.

So sieht Deine webbasierte Entwicklungsumbegung aus. Zu Beginn jedes Levels werden Dir zwei Code-Editoren angezeigt.

In der Ersten wird die Struktur der Daten in JavaScript-Code erläutert. Es werden nicht die eigentlichen Datensätze dargestellt, denn wie der Name "Big Data" schon andeutet, handelt es sich hier um ganz viele Daten, die nur automatisiert mit klugen Algorithmen ausgewertet werden können.
In diesem Spiel ist die Datenbank immer ein JavaScript-Objekt. So kannst Du bequemer auf die Daten zugreifen.
Wie Du sehen kannst, sind die Daten in der Variable fruits, welche ein Array ist, gespeichert.

Die eigentliche Arbeit wird im zweiten Code-Editor verrichtet. Denn in den kannst Du Dein Programm eintippen.
Und sobald Du der Meinung bist, dass Dein Programm fertig geschrieben ist, solltest Du auf den Button Ausführen klicken, um Dein Programm auszufuhren.

Während des Ausführens klappt weiter unten eine Konsolenausgabe auf und neben dem Ausführen-Button ein Lösung zeigen-Button auf.
In der Konsolenausgabe wirst Du alle Meldungen Deines Programms sehen, die Du mit console.log aufgerufen hast. Falls Aber Dein Programm fehler aufweist, wird eine entsprechende Bemerkung in der Konsolenausgabe angezeigt. Es ist natürlich nicht schlimm, wenn Dein Code Fehler enthält. Jeder gute Informatiker baut unwissentlich regelmäßig Fehler in den Programm-Code ein. Es heißt ja auch, ein guter Informatiker ist keiner, der auf Anhieb fehlerfreien Code schreibt, sondern jemand, der enthaltene Fehler wieder beseitigen kann.

Falls Du doch nicht auf die Lösung kommst, kannst Du nach der ersten Ausführung jeder Zeit Dir die Lösung anzeigen lassen. Dazu musst Du einfach den Lösung zeigen-Button anklicken.
Aber Du sollst wissen, die angezeigte Lösung wird immer nur EIN möglicher Algorithmus von VIELEN sein, die alle das gleiche Problem lösen.

Die Anweisung für das erste Level ist:
Gebe das Array fruits elementweiße mit console.log aus.
var fruits = [];
var fruits = ["apple", "pear", "grape", "banana"]
fruits.forEach(console.log);
